# Complex Proficiencies
### Adds proficiencies that allow for players to level up and increase their ability to do a task.
A list of proficiencies currently added include:
```
-Weaponsmithing
-Armorsmithing
-Cooking
-Fishing
-Leatherworking
-Mining
-Handicrafts
-Metalworking
```
Features include:
```
-Level gating
-Bonus stats/drops
-Custom crafting
-NBT support
```
